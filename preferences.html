General
Enable Subtitle Rotation
Automatically rotate subtitle text to match the video rotation angle.
Adjust Subtitle Position
Repositions subtitles to the visual bottom of the rotated video.
Disable if subtitles jump to unexpected positions.
Show OSD on Rotation Change
Display a brief on-screen message when the subtitle rotation is updated.
Advanced
ASS Override Mode
Yes – recommended for most content. Preserves original ASS styling; only the Default style is rotated.
Force – forces all subtitle text through the Default style. Use if subtitles use custom style names and rotation is not applied. Note: loses original color, size, and position overrides in complex ASS files.
Scale – minimal override; may not apply angle changes correctly.
